### Цель
Создать агента, который берет на себя рутину скрам-мастера

### Метрики
Продуктовая: % задач, которые пользователь аппрувит без редактирования (Цель > 80%)
Техническая: Успешное сведение графа сущностей без дубликатов на текстах > 20к токенов. Удержание p95 latency агента в пределах 30 секунд

### Сценарии использования
Агент видит "атомы" типа task, обогащает их контекстом и предлагает завести тикеты.
QA по истории: Юзер спрашивает: "С какими трудностями сталкивался Имя за последний месяц?". Агент использует Tool для поиска по векторной базе атомов и генерирует ответ.

### Архитектурный набросок
Data Layer: FastAPI + RabbitMQ + Deepgram + LLM Map-Reduce Extractor
Agent Layer: LangChain/LangGraph агент с инструментами (Trello API, DB Vector Search)
State: PostgreSQL
