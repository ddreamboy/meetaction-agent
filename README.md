## MeetAction Agent (PoC)
### Что за задача
Автоматизация менеджмента встреч. Сейчас после созвонов нужно вручную извлекать договоренности и заводить задачи. Боль - потеря контекста и трата времени (до 4 часов в неделю)

### Что сделает PoC на демо
Пользователь загружает аудио/видео рабочей встречи. Система извлекает граф знаний (кто, что, кому обещал). Затем Агент анализирует граф, самостоятельно формирует список тасок, предлагает их пользователю и, получив "ОК", создает карточки в тестовом Trello/Notion через API. Также агент может ответить на вопросы по базе прошлых встреч

### Что НЕ делает (out-of-scope)
Не интегрируется с Zoom/Meet в реальном времени (работает с записями). Не поддерживает распознавание эмоций
